Guía: Conectar Google Sheets
  1. Entrá a console.cloud.google.com
  2. Creá un proyecto nuevo (ej: "EscanéaTicket")
  3. Activá la API: Google Sheets API
  4. Ir a Credenciales → Crear credencial → ID de cliente OAuth
  5. Tipo de aplicación: Aplicación web
  6. En "Orígenes autorizados" agregá: null (para archivos locales)
  7. En "URIs de redireccionamiento" agregá: https://escaneaticket.local — y también añadí en la pantalla de consentimiento tu email como usuario de prueba
  8. Copiá el Client ID y pegalo arriba
✅ Solo necesitás hacer esto una vez. El token se guarda en el navegador.
